职位名称：LLM 应用工程师

岗位职责：
1. 负责大模型应用系统的设计、开发与上线，支持招聘、知识问答等业务场景。
2. 基于 FastAPI 构建后端服务，完成接口设计、性能优化与稳定性保障。
3. 参与 Prompt Engineering、RAG 方案设计和效果评估，持续提升回答质量。
4. 与产品和业务团队协作，将需求转化为可落地的 AI 功能。

任职要求：
1. 熟悉 Python 开发，具备良好的工程实现能力。
2. 了解 LLM 应用开发流程，具备 Prompt 调优经验。
3. 熟悉 RESTful API 设计，有 FastAPI 或 Flask 项目经验。
4. 具备良好的沟通能力和跨团队协作能力。

加分项：
1. 有向量数据库、RAG、Agent 相关项目实践。
2. 有求职、教育、HR 科技相关场景经验优先。
